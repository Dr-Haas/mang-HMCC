# Dossier vidéos

Placez votre vidéo d'introduction ici.

**Nom recommandé** : `intro.mp4`

**Format recommandé** : MP4 (H.264) pour une meilleure compatibilité

**Taille recommandée** : 
- Résolution : 1920x1080 (Full HD) ou 1280x720 (HD)
- Durée : 15-30 secondes idéalement
- Poids : < 5 MB si possible (compression recommandée)

Une fois la vidéo ajoutée, elle sera automatiquement utilisée par le composant `VideoLoader`.
